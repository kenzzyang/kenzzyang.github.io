# Ken Yang Blog

基于 `Astro` + `Fuwari` + `Pagefind` 的双语个人博客，目标部署到 `GitHub Pages`。

- 目标地址：`https://kenzzyang.github.io/`
- 仓库建议：`https://github.com/kenzzyang/kenzzyang.github.io`

## 本地开发

```powershell
corepack enable
pnpm install
pnpm dev
```

## 发布

推送到 `main` 后，可通过 GitHub Actions 自动构建并部署到 Pages。
